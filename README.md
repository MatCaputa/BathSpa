## Assessments

### Programming Skills

- **[CodeLab(AS1)](CodeLab/CodeLab(AS1)/)** — *Programming Skills Portfolio.* Ten standalone C++ programs covering output streams, primitive types, conditionals, `switch`, loops, arrays and functions. See the [README](CodeLab/CodeLab(AS1)/README.md) for the full exercise list.

### Databases

- **[Database(AS2)](Database/Database(AS2)/)** — *STEM Historical Figures Database.* A fully normalised (3NF) MySQL schema for historical STEM figures and an event-booking extension, plus an equivalent MongoDB implementation. See the [README](Database/Database(AS2)/README.md).

### Web Development

- **[WebDev(AS1)](WebDev/WebDev(AS1)/)** — *Set Exercises.* Four standalone web pages demonstrating semantic HTML, CSS styling, page layout, and accessibility. See the [README](WebDev/WebDev(AS1)/README.md).
- **[WebDev(AS2)](WebDev/WebDev(AS2)/)** — *Web Dev Project.* End-to-end client-brief website with HTML, CSS, JavaScript, multiple pages and rich media. See the [README](WebDev/WebDev(AS2)/README.md).
